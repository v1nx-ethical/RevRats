STARTUP:
1. MAKE SURE YOU HAVE ROBLOX COMPLETELY CLOSED

2. RUN "RoFlag.exe" 

3. OPEN ROBLOX

4. PRESS "Q" ON YOUR KEYBOARD

5. THE MENU SHOULD APPEAR

6. PASTE SCRIPT YOU WANT TO USE IN

7. JOIN GAME

8. CLICK LOAD

9. ENJOY!


COMMON ERRORS & FIXES: 
1. DLL NOT INITIALIZED
	FIX - MAKE SURE YOU HAVE ALL THE DLL'S IN THE FOLDER

2. ERROR: PROCESS FAILURE
	FIX - TRY RUNNING AS ADMIN, IF THAT DOESNT WORK, TRY DISABLING 3RD PARTY ANTIVIRUS

3. DOWN FOR MAINTENCE
	FIX - BE PATIENT, THE DEVS ARE UPGRADING THE SOFTWARE TO MAKE IT AS GOOD AS POSSIBLE!

4. OTHER ERRORS
	FIX - CONTACT ADMIN@ROFLAG.CC VIA EMAIL AND PASTE THE ERROR CODE
